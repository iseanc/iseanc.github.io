A Web Development Blog

This is a first entry on the "Web Development Blog" web site. It is not my first web page, but I'm building it from the MDN Web Docs "Getting started with the web" tutorial. I don't know how far this will take me in getting a full front-end page with a back-end (db, etc) as well.

I want the full site to consist of

  • A main page built with HTML, styled with CSS, and some active JavaScript.
  • Maybe one more secondary pages linked to from the main page.
  • Maybe a Sign-Up button so people can create their own blog account.
  • Maybe a Log In button so those with accounts can log in to create entries.
  • A database running on a back-end to store user accounts, blog entries, and whatever else gets created as records.
